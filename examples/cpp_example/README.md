# C++ Example (Core API)

This example demonstrates:
- Creating an in-memory relational storage
- Defining a table schema
- Inserting a few rows
- Selecting and printing results

## Build

From the repo root:

```bash
cmake -S . --preset debug
cmake --build --preset debug -j
```

## Run

```bash
./build/debug/bin/kadedb_cpp_example
```
